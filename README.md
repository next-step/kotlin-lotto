# kotlin-lotto

## 2단계 - 로또(자동)
구현할 기능 목록
 - [x] 로또 구입 금액 입력 받기
 - [x] 로또 구매 횟수 구하기
 - [x] 로또 번호는 1~45 숫자만 가능하다
 - [x] 로또 번호 생성하기
 - [x] 로또 1 게임은 중복 없이 6개의 로또 번호로 이루어진다
 - [x] 로또 번호는 오름차순으로 정렬되어있다
 - [x] 생성된 로또 번호 출력하기
 - [x] 로또 당첨 번호 입력받기
 - [x] 당첨 여부(얼마나 일치하는지) 판단하기
 - [x] 당첨 통계내기
 - [x] 당첨 통계 출력하기
 - [x] 로또 게임 실행하기
 - [x] 2등을 위한 보너스 번호를 입력 받기
 - [x] 보너스 번호는 당첨 번호에 포함되지 않은 번호다
 - [ ] 당첨 통계에 2등도 추가하기
