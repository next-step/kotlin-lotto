# 기능 요구사항
- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의 가격은 1000원이다.


# 작업 사항
- [X] 입력된 구입 금액만큼 로또를 n개 구입한다.
  - [X] 입력된 구입 금액이 1개를 구매할 수 있는 최소 금액이 넘지 않으면 에러처리한다.
  - [X] 입력값이 숫자가 아니면 에러처리한다.
- [X] 구매한 n개를 출력한다.
- [ ] 지난주 당첨 번호를 입력한다.
- [ ] 입력된 지난주 당첨 번호를 토대로 당첨 여부를 계산한다.
- [ ] 당첨 통계와 수익률을 출력한다.
