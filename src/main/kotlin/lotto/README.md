# 로또

### 기능 요구사항

- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야한다.
- 로또 1장의 가격은 1000원이다.
- 입력한 금액, 자동 생성 숫자, 수동 생성 번호를 입력하도록 해야 한다.

### 요구사항 정리

- [x] 자동 로또를 구입할수 있다.
- [x] 로또 숫자의 범위는 1부터 45까지이다.
- [x] 로또 구입 금액을 입력할수 있다.
- [x] 당첨 로또를 생성할수 있다.
- [x] 당첨 통계를 확인할수 있다.
- [x] 로또 2등(5개 숫자 일치, 보너스 숫자 일치)을 추첨할수 있다.
- [ ] 수동 로또를 구입할수 있다.
- [ ] 수동 로또 구입 수, 번호를 입력받을수 있다.
